How to Generate Documentation

Prerequisites: You must be inside the OpenSearch Dashboards container.

  1. Attach to the container (VS Code or docker exec -it osd bash)
  2. Navigate to the plugin directory: 
    cd /home/node/kbn/plugins/custom_plugin
  3. Install TypeDoc if needed: 
    yarn install
  4. Generate the documentation: 
    yarn docs:generate
  5. Refresh this page to view the generated documentation

What Gets Documented

TypeDoc will generate comprehensive API documentation for:

  • Core Configuration - Plugin constants, API paths, defaults
  • Domain Types - Todo entities, enums, statistics types
  • Data Transfer Objects - Request/response contracts for all API endpoints
